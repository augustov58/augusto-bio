# Augusto Valbuena — Personal Bio & Portfolio Site

## Design Reference
PostHog.com — warm, playful, credible. Cream/warm backgrounds, bold sans-serif headings, soft illustrations, hand-drawn feel where appropriate.

## Tech Stack
- Next.js 14 (App Router)
- Tailwind CSS
- TypeScript
- Deploy to Vercel

## Color Palette (PostHog-inspired warm tones)
- Background: `#EEEFE9` (warm off-white/cream)
- Surface cards: `#FFFFFF`
- Primary text: `#1D2939` (dark charcoal)
- Secondary text: `#475467`
- Accent/CTA: `#F54E00` (PostHog orange-red)
- Accent hover: `#E04400`
- Borders: `#D0D5DD`
- Code/mono background: `#F9FAFB`

## Typography
- Headings: `Inter` or `Plus Jakarta Sans` — bold, clean
- Body: `Inter` — readable, modern
- Mono: `JetBrains Mono` — for code snippets

## Key Design Elements
- Rounded corners on cards (12-16px)
- Subtle shadows on hover
- Smooth scroll between sections
- Animated entrance for cards (fade-in on scroll)
- Dark mode toggle (optional, nice-to-have)
- SVG decorative elements (squiggly lines, dots) like PostHog uses

## Content Tone
